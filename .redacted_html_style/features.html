Features

Space Station OS includes the following key features:

  • Advanced Fault Detection, Isolation, and Recovery (FDIR) tools.
  • Real-time simulation and control for space station scenarios.
  • Support for ROS 2-based robotics systems.
  • Expandable architecture for community-driven development.
